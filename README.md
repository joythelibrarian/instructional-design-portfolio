# Joy Kim — Instructional Design Portfolio

A professional portfolio site showcasing eLearning projects, instructional design artifacts, and 15+ years of teaching experience translated into the language of corporate L&D. Built as a static site and hosted free on GitHub Pages.

**Live site:** https://joythelibrarian.github.io/id-portfolio

---

## About this site

This repository contains the source for my instructional design portfolio. The site is built with plain HTML and CSS — no frameworks, no build step — so it's fast, easy to maintain, and easy for anyone to inspect.

The site is organized around five sections:

1. **About** — My journey from higher education classrooms to instructional design, IDOL Academy internship, and core competencies
2. **eLearning Projects** — Featured case study: *Beat the Bot: Building Your AI Career Agent*
3. **Process Artifacts** — Storyboards, design documents, workbooks, and scripts demonstrating my full ID workflow
4. **Teaching Experience** — 15+ years across higher education and K-12 in the Bay Area
5. **Contact** — How to reach me

## Featured Project: Beat the Bot

A task-centered eLearning workshop built during my IDOL Academy internship. The course helps IDOL students close the "employability gap" by building a custom *AI Career Agent* — a persistent chatbot trained to act as their personal recruiter, copywriter, and interview coach. Reframes the opaque hiring process as a "Firewall" the learner is engineered to penetrate.

Full design artifacts include:
- Instructional Design Document
- Design Workbook Template
- Storyboard (v3)
- Course Script (v1)

Built using ADDIE & SAM frameworks with SME collaboration.

## How the site is deployed

This site lives at `joythelibrarian.github.io/id-portfolio` as a GitHub Pages project site under my main GitHub account.

To deploy updates:

1. Edit `index.html`, `style.css`, or `README.md` locally
2. Open GitHub Desktop — it shows what changed
3. Add a short commit message describing the change
4. **Commit to main**, then **Push origin**
5. Changes appear on the live site within about a minute

## Design

The site uses an editorial, refined aesthetic:

- **Typography:** Cormorant Garamond (display serif) paired with Inter Tight (body sans) and JetBrains Mono (technical labels)
- **Palette:** Deep navy ink (`#15202b`) on warm ivory paper (`#f6f1e8`) with an ember-red accent (`#a13e2a`)
- **Layout:** Generous whitespace, sharp typographic hierarchy, asymmetric grids, subtle scroll-reveal animations

The aesthetic is intentional — it signals familiarity with information design and editorial craft, which matters for instructional design roles.
